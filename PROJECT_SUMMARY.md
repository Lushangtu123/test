## 🚀 如何使用

### 1. 本地运行

**方式一：直接打开**
```bash
# 用浏览器直接打开
open index.html
```

**方式二：本地服务器**
```bash
# Python 服务器
python3 -m http.server 8000

# Node.js 服务器
npx http-server -p 8000

# 访问
open http://localhost:8000
```

### 2. 在线部署

**GitHub Pages**
```bash
# 1. 创建 GitHub 仓库
# 2. 上传所有文件
# 3. 在仓库设置中启用 GitHub Pages
# 4. 访问：https://yourusername.github.io/webcode-studio
```

**Netlify**
```bash
# 1. 注册 Netlify 账号
# 2. 连接 GitHub 仓库或直接上传文件
# 3. 自动部署完成
```
